Featured Project — TED Platform

TED E-Invoicing Platform


Regulatory E-Invoicing & Tax Compliance Platform

A full-scale enterprise platform built to enable businesses comply with Nigeria’s NRS e-invoicing requirements under the Merchant Buyer System (MBS).


Problem

Businesses lacked infrastructure to generate standardized invoices, validate them, and transmit securely to regulatory systems.


What I Built

  • Full-stack web platform for invoice generation and compliance
  • API integration system for ERP/accounting software
  • Secure transmission pipelines to regulatory endpoints
  • Invoice validation engine with IRN generation
  • Batch processing via Excel/CSV uploads
  • Job queue system for controlled submissions
  • Wallet and token-based billing system
  • Dashboard for tracking invoice status and activity

Impact

  • Enabled SMEs and enterprises achieve compliance
  • Reduced manual tax reporting workload
  • Provided scalable infrastructure for high-volume transactions
